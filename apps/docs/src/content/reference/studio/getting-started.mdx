### Installation

```bash copy
npm install @threlte/studio
```

### Quick Start

To get started, encapsulate your whole scene in the component [`<Studio>`](./studio).

```svelte title=App.svelte {3,8,10}+
<script lang="ts">
  import { Canvas } from '@threlte/core'
  import { Studio } from '@threlte/studio'
  import Scene from './Scene.svelte'
</script>

<Canvas>
  <Studio>
    <Scene />
  </Studio>
</Canvas>
```

To use auto-sync, in your vite config, insert the Threlte Studio vite plugin before any other plugin.

```js title=vite.config.js {2}+ {5}m
import { sveltekit } from '@sveltejs/kit/vite'
import { studioPlugin } from '@threlte/studio/vite'

export default {
  plugins: [studioPlugin(), sveltekit()]
}
```
