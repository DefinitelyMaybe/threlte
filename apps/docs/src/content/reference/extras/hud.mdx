Renders a heads-up-display (HUD). Each HUD creates a new scene rendered on top of the main scene with a separate Threlte context and camera. 

<Tip type="warning">
  The HUD reuses parts of the main Threlte context that wouldn't be useful to duplicate, such as the renderer and the task scheduler.
</Tip>

<Example path="extras/hud" />

### Examples

#### Basic Example

```svelte
<script lang="ts">
  import { T } from '@threlte/core'
  import { HUD } from '@threlte/extras'
</script>

<HUD>
  <T.PerspectiveCamera makeDefault>
  <T.Mesh>
    <T.BoxGeometry />
    <T.MeshStandardMaterial />
  </T.Mesh>
</HUD>
```
