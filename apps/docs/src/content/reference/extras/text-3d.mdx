Render 3D text using threejs's [TextGeometry](https://threejs.org/docs/index.html#examples/en/geometries/TextGeometry).

<Example path="extras/text-3d" />

## Examples

### Basic Example

```svelte title="Text3D.svelte"
<script lang="ts">
  import { T } from '@threlte/core'

  import { Text3D } from '@threlte/extras'
</script>

<Text3D font={'path-to-your-font'} text={'Hello'}>
	<T.MeshStandardMaterial />
</Text3D>
```

### Loading a font yourself

You can also load the font yourself and pass it to the component, like so:

```svelte title="Text3D.svelte"
<script lang="ts">
	import { T, useLoader } from '@threlte/core'
	import { Text3D } from '@threlte/extras'

  import { FontLoader } from 'three/examples/jsm/Addons.js'

	let font = useLoader(FontLoader).load('path-to-your-font')
</script>

{#await $font then loadedFont}
	<Text3D font={loadedFont} text={'Hello'}>
		<T.MeshStandardMaterial />
	</Text3D>
{/await}
```

### Generate a font

You can generate a font using [typeface.js](https://gero3.github.io/facetype.js/).
